📌 ШАГ 1: ПОЛУЧИ ТОКЕН БОТА
═══════════════════════════════════════════════════════════════════════════

1. Открой Telegram
2. Найди @BotFather (поиск, официальный аккаунт Telegram)
3. Отправь команду: /newbot
4. Следуй инструкциям BotFather:
   - Введи имя бота (например: "MyAwesomeBot")
   - Введи username бота (например: "my_awesome_bot")
5. BotFather отправит тебе токен вида:
   123456789:ABCDefGHIjklmnoPQRstuvWXYZ-aBcDeFGhI
6. Скопируй этот токен


📌 ШАГ 2: НАСТРОЙ .ENV ФАЙЛ
═══════════════════════════════════════════════════════════════════════════

1. Открой файл .env (находится в папке сботом)
2. Найди строку: BOT_TOKEN=YOUR_BOT_TOKEN_HERE
3. Замени на:      BOT_TOKEN=123456789:ABCDefGHIjklmnoPQRstuvWXYZ-aBcDeFGhI
                   (Используй ТВОЙ токен от BotFather!)
4. Сохрани файл


📌 ШАГ 3: УСТАНОВИ ЗАВИСИМОСТИ
═══════════════════════════════════════════════════════════════════════════

Вариант A (Автоматический на Windows):
   → Двойной клик на: install_requirements.bat
   
Вариант B (Ручная установка в PowerShell/CMD):
   pip install -r requirements.txt
   
Вариант C (Установка пакетов по отдельности):
   pip install aiogram
   pip install python-dotenv
   pip install gspread


📌 ШАГ 4: ЗАПУСТИ БОТА
═══════════════════════════════════════════════════════════════════════════

Вариант A (Автоматический на Windows):
   → Двойной клик на: run_bot.bat
   
Вариант B (Командная строка):
   python simple_bot.py        (Простой бот - рекомендуется!)
   или
   python bot.py               (С Google Sheets)


📌 ШАГ 5: ПРОТЕСТИРУЙ БОТА
═══════════════════════════════════════════════════════════════════════════

1. Найди своего бота в Telegram по username (например: @my_awesome_bot)
2. Отправь команду: /start
3. Выбери одно из предложенных состояний
4. Получи ответ! 🎉

   "😊 Новое состояние": "Твой ответ здесь",

╔════════════════════════════════════════════════════════════════════════════╗
║                        ❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ                         ║
╚════════════════════════════════════════════════════════════════════════════╝

❓ Бот не запускается?
   1. Проверь, правильный ли токен в .env
   2. Проверь интернет
   3. Убедись, что установлены все пакеты: pip install -r requirements.txt
   4. Перезагрузи программу

❓ Как изменить текст ответов?
   Открой simple_bot.py, найди CARD_DATA и отредактируй текст

❓ Как добавить новую кнопку?
   Добавь строку в CARD_DATA:
   "😊 Новое состояние": "Текст ответа",

❓ Как добавить ссылку/сайт в ответ?
   Используй HTML теги:
   "😊 Состояние": "<a href='https://example.com'>Мой сайт</a>",

❓ Как добавить пустую строку в тексте?
   Используй \n:
   "Строка 1\nСтрока 2\nСтрока 3"

❓ Как отправить бота на сервер?
   • VPS (например, DigitalOcean, Linode)
   • Google Cloud, AWS, Heroku
   • Домашний компьютер (24/7 должен быть включен)
   • Telegram Cloud Storage (если использовать webhook)
