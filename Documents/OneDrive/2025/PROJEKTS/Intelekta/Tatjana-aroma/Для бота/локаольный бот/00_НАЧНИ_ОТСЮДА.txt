🎯 ПЕРВЫЕ 3 ШАГА:

   1️⃣  ТОКЕН
       Откройте Telegram → найдите @BotFather → /newbot
       Получите токен вида: 123456:ABC...

   2️⃣  .ENV
       Откройте файл .env, вставьте:
       BOT_TOKEN=123456:ABC...

   3️⃣  ЗАПУСК
       pip install -r requirements.txt
       python simple_bot.py
       
   ГОТОВО! 🎉
