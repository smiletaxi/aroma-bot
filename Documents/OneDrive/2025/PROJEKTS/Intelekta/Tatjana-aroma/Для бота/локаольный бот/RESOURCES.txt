📝 ШПАРГАЛКА ПО КОМАНДАМ
───────────────────────────────────────────────────────────────────────────

PowerShell / CMD:

# Установить пакет
pip install имя_пакета

# Обновить пакет
pip install --upgrade имя_пакета

# Удалить пакет
pip uninstall имя_пакета

# Посмотреть установленные пакеты
pip list

# Запустить Python скрипт
python имя_скрипта.py

# Запустить Python интерпретатор
python

# Создать виртуальное окружение
python -m venv venv

# Активировать виртуальное окружение (Windows)
venv\Scripts\activate

# Деактивировать виртуальное окружение
deactivate

🎯 РЕКОМЕНДУЕМАЯ ПОСЛЕДОВАТЕЛЬНОСТЬ ОБУЧЕНИЯ
───────────────────────────────────────────────────────────────────────────

1️⃣  Изучи основы Python:
   - Переменные, типы данных
   - Функции и циклы
   - Работа со списками и словарями
   - Обработка ошибок

2️⃣  Изучи асинхронное программирование:
   - async/await
   - asyncio
   - Зачем нужна асинхронность

3️⃣  Начни с простого бота:
   - Запусти simple_bot.py
   - Добавь новые состояния
   - Экспериментируй

4️⃣  Изучи aiogram документацию:
   - Хэндлеры
   - Фильтры
   - Клавиатуры
   - Состояния

5️⃣  Добавляй сложные функции:
   - Базу данных
   - Платежи
   - AI интеграцию
   - Вебхуки

6️⃣  Задеплой бота:
   - На сервер
   - В облако
   - На Heroku/DigitalOcean

❓ ЧТО ДАЛЬШЕ?
───────────────────────────────────────────────────────────────────────────

Когда простой бот заработает:

1. Изучи примеры в файле examples.py
2. Добавь новые состояния и ответы
3. Запусти бота в Telegram и тестируй
4. Когда все стабильно - добави в Telegram Store
5. Приглашай людей в бота
6. Собирай отзывы и улучшай
