🚀 САМЫЙ БЫСТРЫЙ СТАРТ:

1. Открой START_HERE.txt (3 минуты чтения)

2. Или просто выполни эти шаги:

   a) Установи зависимости:
      pip install -r requirements.txt

   b) Получи токен:
      - Открой Telegram
      - Найди @BotFather
      - Отправь /newbot
      - Получи токен

   c) Вставь токен в .env файл:
      BOT_TOKEN=твой_токен_здесь

   d) Запусти бота:
      python simple_bot.py

   e) Найди бота в Telegram и отправь /start

   f) Готово! 🎉
